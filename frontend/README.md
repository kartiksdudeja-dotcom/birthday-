# Noor Birthday Project - Full Stack

A premium, animated birthday experience for Noor, built with React, Framer Motion, and Spring Boot.

## Features
- **Magical UI**: Recreated from the original high-end HTML design with enhanced animations.
- **Tree Blooming**: A beautiful canvas-based animation where hearts bloom to form a larger heart.
- **Typewriter Greetings**: Smooth, sequential text reveals.
- **Surprise Gift**: Interactive gift box that unwraps with a 3D effect.
- **Cloud Reflections**: Save and load personal reflections to a Java Spring Boot backend.

## Tech Stack
- **Frontend**: React 18, Vite, TypeScript, Framer Motion, Lucide Icons, Axios.
- **Backend**: Java 17, Spring Boot 3, Spring Data JPA, H2 Database.

## How to Run

### 1. Start the Backend (Node.js)
Navigate to the `backend` folder and run:
```bash
cd backend
npm install
npm start
```
The API will be available at `http://localhost:8080`.

### 2. Start the Frontend (React)
Navigate to the root folder and run:
```bash
npm install
npm run dev
```
Open `http://localhost:5173` in your browser.

## Project Structure
- `/src`: React frontend source code.
- `/backend`: Spring Boot Java application.
- `code.html`: Original design for reference.
